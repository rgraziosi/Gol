Teste prático C# - API RESTful with Angular 7

1. Desenvolva uma APIs RESTful com a API da Web ASP.NET, no projeto deve seguir as seguintes especificações.

a. Controller Airplane -> Listagem de aeronaves que irão se relacionar com os passageiros.
i. GetAllAirplane();
ii. FindAirplane(id);
iii. InsertAirplane(Airplane request);

b. Controller Passangers -> Listagem de passageiros i. InsertPassanger(Passanger request);
ii. InsertPassangerToAirplane(PassangerToAirplane request);
iii. ChangePassanger(int id, IdAirplane);
iv. ListAllPassangerByAirplane(int idAirplane);

c. Deve ser criada uma conexão com banco de dados (utilizar classe SqlConnection)

d. Desenvolver um projeto de teste, construir teste unitário relacionado às chamadas das controllers.

i. No teste unitário deve ser possível executar todos os métodos das controllers, fincando possível realizar as ações de CRUD pelo teste unitário.


2. Desenvolver um projeto com Angular7, o projeto deve conter 2 controllers com as seguintes funções:

a. Airplane
i. Listar aeronaves.

b. Passangers
i. Adicionar um passageiro.
ii. Alterar aeronave de um passageiro.
